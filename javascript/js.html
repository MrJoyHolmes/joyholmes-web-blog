    # Javascript对象属性（数据属性、访问器属性）

    1. 数据属性：

    [[value]] 它可以是任何的类型，表示这个数据的值。默认值：undefined

    [[writable]] 它是布尔类型，表示这个数据的[[Value]]是否可以被修改。默认值①：false

    [[enumerable]] 它也是布尔类型，表示这个数据能否用for…in循环或Object.keys来枚举。默认值①：false

    [[configurable]] 它依然是布尔类型，表示这个数据能否被delete运算符删除。默认值①：false

    ①这里的默认值是指在Object.defineProperty()中定义属性的时候的默认值 用Object.getOwnPropertyDescriptor()获取数据属性

    
    var o =function(){}; // 创建一个新对象

    // 在对象中添加一个属性与数据描述符的示例
    Object.defineProperty(o, "a", {
        value : 37,
        writable : true,
        enumerable : true,
        configurable : true
    });

    Object.defineproperty(o,"eat",{
        value : function(){
            console.log("eating!!!");
        },
        writable:false,
        enumerable : true,
        configurable : true
    });

    o.a;
    o.eat();
    1. 访问器属性

    [[Get]] 它是函数或者undefined类型

    [[Set]] 它是函数或者undefined类型

    [[enumerable]] 它也是布尔类型，表示这个数据能否用for…in循环或Object.keys来枚举。默认值①：false

    [[configurable]] 它依然是布尔类型，表示这个数据能否被delete运算符删除。默认值①：false

    使用getter、setter来创建访问器

    语法：get functionName(){functionBody}

        var data={
        name:'pad',
        get showName(){return this.name;},
        set setName(name){this.name=name;}
    }
    data.showName;
    data.setName="newName";

    set方法（访问器）只能接受一个参数，并且在调用访问器的时候，不是像调用传统函数funcName(para);一样用括号包裹，而是data.setName="newName";用等号“赋值”

    使用Object.defineProperty来创建访问器

        var data={
        _name:'pad'
    };
    Object.defineProperty(data,"name",{
        get:function(){
            return this._name;
        },
        set:function(newName){
            this._name = newName;
        }
    });

    var person={
        name:'xiaoming'
    }
    Object.defineProperty(person,"sayName",{
        get:function(){
            return this.name;
        }
    });
    person.sayName;

    # js 数组常用方法

    • concat，把多个数组中的值合并到一起。
    • fill(value:c T, start?: number, end?: number)
    • find，查找第一个符合条件的元素，返回值为元素的值；
    • findIndex，查找第一个符合条件的元素，返回值为元素在数组中的索引；参数是一个函数。
    • forEach(callback(currentValue [, index [, array]])[,thisArg])，遍历数组，这种遍历不能暂停，只能遍历到结尾自动结束；
    • includes, 数组中是否包含某个数据；
    • indexOf，某个 item 在数组中的下标，如果下标为 -1 说明数组中不存在这个元素，常用这个判断数组中是否包含某个元素；
    • lastIndexOf，某个 item 在数组中的下标，如果下标为 -1 说明数组中不存在这个元素，从末尾开始遍历；
    • map，遍历数组中的元素，重新组成一个数组；
      const array1 = [1, 4, 9, 16];

  // pass a function to map
  const map1 = array1.map(x => x * 2);

  console.log(map1);
  // expected output: Array [2, 8, 18, 32]
    • slice(beg, end)，取数组中某个范围内的元素组合成一个新的数组，不会改变原数组；
    • splice(start[, deleteCount[, item1[, item2[, ...]]]])，删除或插入元素到指定的位置。在数组第二个位置插入元素 drum，不进行删除。
    let myFish = ['angel', 'clown', 'mandarin', 'sturgeon']
let removed = myFish.splice(2, 0, 'drum')

// myFish is ["angel", "clown", "drum", "mandarin", "sturgeon"] 
// removed is [], no elements removed
    • some 查询数组中是否有符合某个条件的元素，比如查看数组是否存在偶数。
      const array = [1, 2, 3, 4, 5];

  // checks whether an element is even
  const even = (element) => element % 2 === 0;

  console.log(array.some(even));
  // expected output: true

    # 对象的遍历

    # for...in

    遍历了自身和继承的所有可枚举属性；不包含不可枚举和Symbol属性

    # Object.keys

    Object.keys执行后会返回一个数组，而且只遍历了自身可枚举的属性；不包括不可枚举和Symbol属性

    # Object.getOwnPropertyNames

    Object.getOwnPropertyNames执行后返回一个数组，遍历了自身的可枚举属性和不可枚举属性；不包含Symbol属性

    # Reflect.ownKeys

    Reflect.ownKeys执行后返回一个数组，遍历了自身所有属性，包括可枚举、不可枚举、Symbol属性

    # Object.getOwnPropertySymbols

    Object.getOwnPropertySymbols执行后返回一个数组，遍历了对象自身的所有Symbol属性

    # 深拷贝

    1. 定义一个深拷贝
      function deepClone(obj) {
      if (obj === null) return obj;
      if (typeof obj !== "object") return obj;
      if (obj instanceof RegExp) return new RegExp(obj);
      if (obj instanceof Date) return new Date(obj);
      let result = {}; // 初始化返回结果
      if (obj instanceof Array) result = [];

      for (const key in obj) {
          // 保证 key 不是原型属性
          if (obj.hasOwnProperty(key)) {
              // 递归调用
              result[key] = deepClone(obj[key]);
          }
      }
      return result;
  }

function deepClone(val) {
  return new Promise(resolve => {
    const { port1, port2 } = new MessageChannel()
    port2.onmessage = e => resolve(e.data)
    port1.postMessage(val)
  })
}

    # 数组的组合问题

    给定一个数组 arrays = [["a", "b"],["n", "m"],["0", "1"]]，请输出结果[ [a,n,0], [a,n,1], [a,m0,], ... [b,m,1] ]

    1. 循环思路
    // 循环的方式求数组的组合
function combination(arr) {
  let result = [];
  if (arr.length >= 1) {
    arr[0].forEach((item) => result.push([item]));
    for (let i = 1; i < arr.length; i++) {
      const pre = result;
      result = [];
      pre.forEach((list) => {
        arr[i].forEach((item) => result.push([...list, item]));
      });
    }
    return result;
  }
}

// 使用reduce
function combination2(arr) {
  const result = arr.reduce((pre, cur) => {
    if (pre.length === 0) return cur.map((item) => [item]);
    let preList = [...pre];
    let newList = [];
    preList.forEach((list) => {
      cur.forEach((item) => newList.push([...list, item]));
    });
    return newList;
  }, []);
  return result;
}

// 使用递归

function combination3(arr) {
  let result =  [];
  const dps = (curArr, i) =>{
    if(i >= arr.length){
      result.push([...curArr]);
      return;
    }
    arr[i].forEach((item) => dps([...curArr,item], i+1));
  }
  dps([], 0);
  return result;
}
